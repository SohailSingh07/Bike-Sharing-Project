# 🚲 Bike Sharing Demand Prediction

## 📌 Project Overview

This project predicts bike rental demand using Machine Learning.
It helps optimize bike availability and improve user experience in rental systems.

---

## 🧠 Tech Stack

* Python
* Jupyter Notebook
* Scikit-learn
* Pandas, NumPy
* Matplotlib / Seaborn

---

## 📊 Dataset

* Contains information like date, temperature, humidity, windspeed, and rentals count
* Used for training and testing prediction models

---

## ⚙️ Models Used

* Linear Regression
* Decision Tree
* Random Forest (best performing)

---

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Open the notebook:

   ```bash
   jupyter notebook
   ```

---

## 📈 Results

* Random Forest gave the best accuracy
* Model predicts demand effectively for unseen data
